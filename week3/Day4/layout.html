Main Content Area

This section demonstrates the CSS Box Model, which consists of:

Content: The actual content of the element (text, images, etc.)

Padding: Space inside the border, around the content. Applied using padding property.

Border: A line that surrounds the padding. Applied using border property.

Margin: Space outside the border, between this element and other elements. Applied using margin property.

Each section on this page uses these box model properties to create spacing and visual separation.
